Project 1: studyBuddy


This project is a web page showcasing a studyBuddy business. I utilized HTML, CSS, and Bootstrap to design the structure of the page. I included different fonts, colors, images, and icons to style it. The page is split into different sections including features, testimonials, and the pricing of the studyBuddy app. There are buttons associated with some sections and a navigation bar at the top to bring a user to a specific section. The skills I learned from this project can be applied toward understanding website design for businesses and how to approach marketing for a brand.

Project 2: Sleep App


This project is a prototype sleep app made using Figma. There are four frames for the app: a login screen, a screen displaying different sleep music options, a screen to set an alarm, and a menu overlay screen. I chose to use cool toned pastel colors as the main theme, gray for the icons, and white for some details as a way to contrast from the other colors. Additionally, I added stars to the design to correlate with sleep and the night sky. The icons are linked to different frames in the app. The knowledge I have acquired from this project has helped me become more familiar with app design.

Project 3: Schedule


This project was an assignment made using HTML tables and CSS. It shows a weekly schedule with time intervals. The row listing the time intervals is formatted with a mint color and bold font, while the column listing the days is formatted with a purple color and a dashed border. Some activities include an image and some are linked with a webpage. When the cursor hovers over the links, the shape of the cursor changes, the links are highlighted, and the font color changes to black. This project taught me the basics of CSS, which can be useful when understanding website design for businesses.
